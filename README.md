# FUTURE_CS_01
Internship
Web Application Vulnerability Assessment Internship Task

## Project Overview
This repository contains my cybersecurity internship assignment focused on
performing a vulnerability assessment of the OWASP Juice Shop application
using OWASP ZAP.

## Tools Used
- OWASP ZAP
- OWASP Juice Shop
- Java 17

## Contents
- Security Assessment Report (PDF)
- OWASP Top 10 Mapping
- ZAP Scan Report
- Screenshots of vulnerabilities

## Disclaimer
This project was conducted strictly for educational purposes on an
intentionally vulnerable application.
